프로젝트 설명

Pomodoro 타이머를 이용한 시간관리와 음식점 리뷰 기능이 추가된 개인 웹 프로젝트 입니다. 이 프로젝트는 학습한 내용을 적용하고자 시작되었으며, Pomodoro 시간 관리 기법과 축구에서 가장 뛰어난 선수에게 수여되는 Ballon d'Or 상을 결합하여 목표를 달성하고 가장 뛰어난 사람이 되려는 의미로 명명하였습니다. 시간 관리 페이지를 제작 중에 사용자의 요구에 따라 음식점 리뷰 기능을 확장하게 되었습니다.

Java, Spring Boot, Oracle 18C를 기반으로 제작하였으며, AWS S3를 활용하기 위해 AmazonS3Client 라이브러리를 사용했습니다. 이를 통해 클라우드를 기반으로 리소스를 효과적으로 제공할 수 있었습니다. 도메인을 구매하고 AWS EC2를 활용하여 프로젝트를 배포했습니다. 더 나아가 GitHub-Actions와 Docker를 활용하여 CI/CD 파이프라인을 구축했습니다. 이 프로젝트를 통해 개발의 기획부터 배포 및 자동화까지 경험하며 웹 개발의 전반적인 흐름을 이해했습니다.

페이지 최적화 및 속도 개선을 위해 이미지 변형 라이브러리 Thumbnailator를 사용하여 썸네일을 생성하고, 페이지의 끝이 감지되면 추가 리소스를 동적으로 불러오는 무한 스크롤(infinite scroll)을 구현했습니다. ThymeLeaf, Bootstrap 등을 활용하여 Frontend를 구성했습니다.
